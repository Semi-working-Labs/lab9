## Описание проекта

Сайт представляет собой статическую веб-страницу с описанием возможных направлений путешествий.  
Контент включает текстовые блоки, фотографии и отдельный раздел с цитатами.

Проект создан как учебная работа, направленная на отработку навыков структурированной верстки и организации файлов по методологии БЭМ.

---

## Используемые технологии

- **HTML** — разметка страницы.  
- **CSS** — стилизация элементов.  
- **JavaScript** — функциональная логика для отдельных элементов (например, взаимодействие интерфейса).  
- **БЭМ (Блок-Элемент-Модификатор)** — методология, использованная для структуры файлов и именования классов.
